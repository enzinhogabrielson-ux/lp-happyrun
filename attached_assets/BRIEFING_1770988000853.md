# 🏃 BRIEFING - LANDING PAGE CORRIDA HUMANI TREINAMENTOS

## 📋 VISÃO GERAL DO PROJETO

Sistema completo de inscrição para evento de corrida no Bandeiras Empresarial, Votorantim, desenvolvido para a Humani Treinamentos.

## 📄 ESTRUTURA DA LANDING PAGE (`index.html`)

### Hero Section
**Elementos visuais:**
- Background escuro com gradiente
- Dois círculos decorativos com efeito pulse (um superior direito, outro inferior esquerdo)
- Layout em duas colunas (desktop) / empilhado (mobile)

**Conteúdo textual:**
- Logo: "HUMANI TREINAMENTOS"
- Título principal: "Corrida BANDEIRAS"
- Subtítulo: "Vista sua camisa e participe do maior evento de corrida do Bandeiras Empresarial em Votorantim!"
- Detalhes do evento com ícones:
  - 📍 Local: Bandeiras Empresarial, Votorantim
  - 👕 Inclui: Camisa oficial do evento

### Formulário de Inscrição
**Campos:**
1. **Nome Completo** (obrigatório)
   - Placeholder: "Digite seu nome completo"
   
2. **Telefone/WhatsApp** (obrigatório)
   - Máscara automática: (00) 00000-0000
   - Placeholder: "(00) 00000-0000"
   
3. **Tamanho da Camisa** (obrigatório)
   - Opções: PP, P, M, G, GG, XG
   - Select dropdown com descrições
   
4. **Confirmação de Pagamento** (checkbox obrigatório)
   - Texto: "Confirmo que realizei o pagamento da taxa de inscrição e concordo com os termos do evento"

**Botão de Envio:**
- Texto: "CONFIRMAR INSCRIÇÃO"
- Design: Gradiente turquesa com shadow
- Efeito hover: Elevação + intensificação de shadow

**Feedback:**
- Mensagem de sucesso animada após envio
- Reset automático do formulário após 3 segundos

## 🔐 PAINEL ADMINISTRATIVO (`admin.html`)

### Tela de Login
**Credenciais padrão:**
- **Usuário**: `admin`
- **Senha**: `humani2025`

**Interface:**
- Background dark com elemento decorativo
- Box centralizado com blur effect
- Mensagem de erro animada (shake effect)
- Credenciais visíveis na tela para facilitar acesso

### Dashboard Principal

#### Cards de Estatísticas (3 cards)
1. **Total de Inscrições**
   - Cor: Turquesa (#0ADDC8)
   - Valor dinâmico

2. **Pagamentos Confirmados**
   - Cor: Verde (#10b981)
   - Valor dinâmico

3. **Pagamentos Pendentes**
   - Cor: Amarelo (#f59e0b)
   - Valor dinâmico

#### Tabela de Inscritos
**Colunas:**
- Nome
- Telefone
- Tamanho da Camisa
- Data de Inscrição
- Status (Badge colorido: Verde para "Pago" / Amarelo para "Pendente")
- Ação (Botão para alternar status)

**Funcionalidades:**
- Filtros: Todos / Pagos / Pendentes
- Botão para alternar status de pagamento
- Hover effects nas linhas
- Auto-refresh a cada 30 segundos
- Estado vazio quando não há inscrições

## ⚙️ FUNCIONALIDADES TÉCNICAS

### Armazenamento de Dados
- **LocalStorage** do navegador
- Chave: `inscricoes`
- Formato: Array de objetos JSON

**Estrutura de dados por inscrição:**
```json
{
  "nome": "string",
  "telefone": "string (formatado)",
  "tamanho": "string (PP|P|M|G|GG|XG)",
  "pagamentoConfirmado": "boolean",
  "dataInscricao": "ISO string"
}
```

### Validações
- Todos os campos são obrigatórios
- Máscara de telefone automática
- Checkbox de pagamento obrigatório
- Validação HTML5 nativa

### Responsividade
- **Desktop**: Layout em duas colunas
- **Tablet**: Layout adaptativo
- **Mobile**: Layout empilhado, formulário full-width
- Breakpoints: 968px e 640px

## 📱 FLUXO DE USUÁRIO

### Participante (Landing Page)
1. Acessa a landing page
2. Lê informações do evento
3. Preenche formulário (nome, telefone, tamanho)
4. Confirma checkbox de pagamento
5. Clica em "Confirmar Inscrição"
6. Vê mensagem de sucesso
7. Formulário reseta automaticamente

### Administrador (Painel Admin)
1. Acessa `/admin.html`
2. Faz login (admin / humani2025)
3. Visualiza dashboard com estatísticas
4. Vê lista de todos os inscritos
5. Filtra por status (Todos / Pagos / Pendentes)
6. Alterna status de pagamento conforme necessário
7. Faz logout quando terminar

---

## 🚀 COMO USAR

### Instalação
1. Salve ambos os arquivos HTML no mesmo diretório
2. Abra `index.html` no navegador para a landing page
3. Abra `admin.html` no navegador para o painel admin

### Páginas
- **Landing Page**: `index.html`
- **Painel Admin**: `admin.html`

### Credenciais Admin
- **Usuário**: admin
- **Senha**: humani2025

### Dados
- Os dados são salvos automaticamente no LocalStorage do navegador
- Para limpar dados: Abrir DevTools > Application > Local Storage > Limpar
